·본 이벤트는 KT고객에 한하여 참여 가능하며, 중복 참여가 불가능합니다.
·O/X 선택 및 참여하기 클릭 후 경품을 받으실 번호 입력 시 응모 완료 됩니다.
·스타벅스를 제외한 모든 경품은 당첨 즉시 이용 가능한 쿠폰이 문자로 발송됩니다.
(스타벅스는 이벤트 종료 후 9월 9일 기프티쇼 형태로 일괄 전송됩니다.)

·‘5G 슈퍼플랜 로밍혜택 무료 체험권’ 이용 방법

-반드시 사전에 로밍 차단 서비스를 해지 하고, 체험권을 등록 해야 합니다. 차단 서비스를 선해지하지 않을 경우 쿠폰 등록이 실패 처리됩니다.

(*차단서비스: 음성로밍 완전차단/데이터로밍 완전 차단/데이터로밍 완전차단(MMS가능))

-체험권은 다른 데이터로밍부가서비스와 동시에 이용이 불가합니다.

-5G 슈퍼플랜 요금제 고객은 체험권 등록이 불가합니다.(당첨 시, 양도 가능 )

·’로밍ON 5분 무료 통화 쿠폰’이용 방법

-로밍 통화 이용 당월 등록 시, 로밍ON 지원국가(30개) 내 음성/수발신에 한해 혜택이 적용됩니다.

(*로밍ON지원국가: 일본, 중국, 홍콩, 마카오, 대만, 베트남, 태국, 싱가포르, 인도네시아, 필리핀, 말레이시아, 괌, 사이판, 캄보디아,
미국, 캐나다, 영국, 프랑스, 독일, 스페인, 오스트리아, 체코, 폴란드, 헝가리, 아일랜드, 그리스, 터키, 러시아, 호주, 뉴질랜드)

-그 외 국가에서 이용 시, 음성로밍 통화요즘 594원이 할인됩니다.

·'제주항공 리프레시 포인트' 이용 방법

-제공 받으신 인증번호 등록과 포인트 사용은 제주항공 홈페이지, 모바일Web, App에서만 가능합니다.
-인증번호 등록은 마이페이지>나의코폰정보>쿠폰번호 입력 / 등록 후 사용 바라며, 등록기간 내 등록/인증하지 않으면 재발행 및 포인트 사용 불가합니다.
-자세한 리프레시 포인트 사용 유의사항은 제주항공 고객센터(1599-1500)및 홈페이지 (http://www.jejuair.net)를 통해 반드시 확인 바랍니다.

·'스위트북 포토북' 이용방법

-당첨 문자 하단 링크 클릭 시 쿠폰 번호 확인이 가능하며, 포장 배송비 3,000원은 별도입니다. -문의처: 스위트북 고객센터(02-886-0156)
